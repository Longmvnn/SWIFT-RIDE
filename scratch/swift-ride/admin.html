Authenticate with Supabase

This dashboard expects an authenticated admin user whose Supabase JWT includes app_metadata.role = "admin".

Behavior

New drivers are created through a Supabase Edge Function that calls supabase.auth.admin.createUser with a generated password.

Turning the admin master switch off writes is_active = false and immediately forces is_online = false.

The driver list subscribes to realtime updates on the drivers table, so status changes appear without refreshing.
